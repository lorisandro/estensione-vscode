# FocusFlow - Chrome Extension

**Stay focused and productive.** Block distracting websites, use the Pomodoro timer, and track your focus time.

## Features

✅ **Website Blocker** - Block distracting sites during focus mode
✅ **Pomodoro Timer** - Built-in 25/5 timer with auto-focus mode
✅ **Focus Tracking** - See your daily focus time and streak
✅ **Smart Scheduling** - Auto-enable blocking during work hours
✅ **Gamification** - Earn achievements and build streaks
✅ **Beautiful UI** - Modern, clean interface

## Installation (Development)

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the `FocusFlow` folder

## Usage

1. Click the FocusFlow icon in your toolbar
2. Add websites to block (e.g., facebook.com, twitter.com)
3. Toggle Focus Mode ON
4. Use the Pomodoro timer to stay focused
5. Track your progress!
